# predatorCage
a casing for the Anycubic Predator

#### requirements:
- ~10m of 12x3mm sealing tape
- 3x 100x50x3cm acrylic sheets
- 1x universal saw
- 6x [lower holders]()
- 6x [top holders]()
##### optional:
- sandpaper

#### assembly:
- print the holders (i printed them with 32% infill and they work great)
- [cut acrylic sheets]()
- put everything together
